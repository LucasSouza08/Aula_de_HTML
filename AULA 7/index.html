Aula 07 - Estudo de CSS

A Sigla CSS vem do inglês, Cascading Style Sheets, e é uma linguagem utilizada para estilizar páginas da web. Utilizando esta linguagem podemos definir:

  • Cor de fundo, texto e elementos.
  • Fontes e tamanho das letras.
  • Posicionameto de elementos na página.
  • Efeitos visuais e animações.

O CSS funcionaem conjunto com o HTML, enquanto o HTML diz o que aparece, o CSS como deve aparecer.
